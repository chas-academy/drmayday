# Dr. Mayday - WIP

## Getting started

```
git clone git@github.com:chas-academy/drmayday.git
cd drmayday
npm install
```

See `.env.template` for env-variables.

To start live-compiling ReasonML to Javascript run:
`npm start`
then in another terminal run:
`npm run dev`
to fire up the Express-server.
